
# Basic Calculator

This is a basic stopwatch builted in HTML, CSS, JavaScript and Bulma.\
It's just you, 3 buttons and the time.

```bash

```

Go to the project directory

```bash
  cd Stopwatch
```

Open Index.html on your favorite browser!
